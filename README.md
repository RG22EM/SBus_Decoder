SBus_Decoder
============

S.Bus to PWM/PPM decoder
Страница проекта: http://rc-master.ucoz.ru/publ/19-1-0-87

Позволяет получить с выхода S.Bus до 16 каналов PWM (для подключения сервоприводов или ESC) и/или до двух независимых каналов PPM.

Настройка производится посредством приложения SBUSDecoderCFG для ПК (ОС Windows)
